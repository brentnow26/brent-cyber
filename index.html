# About Cybersecurity

    
Cybersecurity is the practice of protecting systems, networks, and data from digital attacks. Here's the landscape, in plain language.

    

      
🛡️
Confidentiality
Keep data secret — only authorized people can read it. Encryption is your best friend.

      
✅
Integrity
Ensure data isn't tampered with. Hashes and signatures prove nothing changed in transit.

      
⚡
Availability
Keep services up and reachable. Defend against outages and denial-of-service attempts.

      
🧠
The Human Layer
Most breaches start with people. Awareness beats almost any firewall.

# Cybersecurity Tools Showcase

    
Industry-standard tools every defender should recognize. (Educational reference only.)

    

      
NmapNetwork mapper & port scanner for inventorying what's exposed.

      
WiresharkPacket analyzer to inspect exactly what crosses the wire.

      
Burp SuiteWeb app security testing & request interception.

      
MetasploitExploitation framework used for authorized pentests.

      
John the RipperPassword hash auditing to find weak credentials.

      
HashcatGPU-accelerated hash cracking for recovery & auditing.

      
Aircrack-ngWi-Fi security auditing suite.

      
OWASP ZAPOpen-source web vulnerability scanner.
